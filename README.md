## About

I design and ship **production AI systems** — LLM apps, multi-agent workflows, RAG platforms, and FastAPI backends that hold up under real load.

My work sits between **model capability** and **system engineering**:
fine-tuning when needed, retrieval when grounding matters, agents when workflows get complex, and clean APIs when products need to scale.

I optimize for architecture, evaluation, latency, cost, and outcomes — not demos that break in production.
