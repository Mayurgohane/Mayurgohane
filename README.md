## About Me

I'm a **Senior AI & Backend Engineer** building production-ready AI systems and scalable backend services. My work spans LLM applications, multi-agent architectures, RAG pipelines, and high-performance APIs — designed to ship reliably, not just demo well.

I focus on clean system design, measurable outcomes, and AI that holds up in real products.

---

## What I Build

- 🤖 **Multi-Agent Systems** — orchestrated agent workflows for research, automation, and complex task execution
- 📚 **Scalable RAG Applications** — retrieval pipelines with vector search, re-ranking, and grounded generation
- ⚡ **LLM-Powered Backend Services** — FastAPI microservices for chat, document intelligence, and AI tooling
- 🧠 **Production ML Systems** — training, evaluation, deployment, monitoring, and iteration loops
- 🔗 **AI + Backend Integration** — connecting models to databases, queues, auth, and cloud infrastructure
